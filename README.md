##  Featured Project — Airbnb-like Listings API

A production-style backend system inspired by Airbnb, built to handle advanced search and booking workflows.

### Key Features

*  **Advanced Search System**

  * Location-based search (with validation)
  * Multi-filter support (price, guests, amenities, type)
  * Smart handling of invalid queries (returns meaningful errors instead of empty success)

*  **Pagination & Performance**

  * Optimized pagination with metadata
  * Query efficiency using Prisma
  * Scalable structure for large datasets

*  **Search Intelligence**

  * Handles edge cases like non-existent locations
  * Prevents misleading empty responses (returns proper 400 errors when needed)
  * Designed for future ranking & relevance scoring

*  **Authentication System**

  * JWT-based authentication
  * Secure password reset via email
  * Token validation and protected routes

* 🗄 **Database Design**

  * Relational schema with Prisma ORM
  * Listings, users, bookings, and photos relationships
  * Clean, maintainable structure
